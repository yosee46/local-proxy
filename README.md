# local-proxy

local-proxy is a gem to allow us to access local server from internet proxying remote server.

# installation



# usage
### server

```cassandraql
ruby server/main.rb --config_path ./my.config.yml
```

### client

```cassandraql
ruby client/main.rb --config_path ./my.config.yml
```

### configuration file
my.config.yml
